`jj` - A [stream editor](https://github.com/tidwall/jj)  
`jq` - A json processor  
`python -m json.tool`

I like `jq` for pretty printing JSON output, `jj` for making JSON pretty or condensed.

This was really useful when optimizing the search index for this blog.

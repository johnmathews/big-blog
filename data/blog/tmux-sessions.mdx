- List of sessions: `<prefix> s`
- Last session: `<prefix> Z`
- Rename session: `<prefix> $`
- Rename window: `<prefix> ,`

- Previous session: `<prefix> (`

For last session I used the following key bind:

```
bind Z switch-client -l
```
